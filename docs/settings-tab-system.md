# Settings Tab System (Implemented)

## Purpose
Provide account management, business preferences, notifications, security, and logout actions.

## Sections
1. Profile
2. Business Settings
3. Notifications
4. Security
5. About
6. Logout

## Row Styling
- Height: `60px`
- Padding: `16px`
- Radius: `20px`
- Background: `#FFFFFF`
- Chevron: `#9CA3AF`

## Color System
### Primary
- Primary Blue: `#2563EB`
- Success: `#16A34A`
- Danger: `#DC2626`
- Muted Text: `#6B7280`
- Border: `#E5E7EB`

## Section Icons
- Profile: Blue
- Business: Blue
- Notifications: Blue
- Security: Blue
- About: Blue
- Logout: Red

## Typography
- Section Title: `16px / 700`
- Row Title: `15px / 600`
- Secondary Value: `14px / 500`

## Layout Rules
- Page background: `#F4F6F8`
- Card radius: `24px`
- Section spacing: `24px`
- Row spacing: `12px`

## Business Settings
### Currency
- Default: `₹ INR`

### Overdue Period
- Default: `7 days`

## Notifications
### Overdue Reminders
- Default: Enabled
- Toggle color: `#16A34A`

## Logout Button
- Background: `#FEF2F2`
- Text: `#DC2626`
- Icon: `#DC2626`

## Currency Rule
Use the `₹` symbol everywhere.

## Approved Interaction
- Tap row → Open corresponding settings screen
- Toggle reminders directly from the list
- Logout requires confirmation dialog
