# User Reports Color System (Approved)

## Purpose
Standardize the visual styling of the user-facing Reports tab so it matches the new dashboard redesign.

## Page Background
- Page background: `#F4F6F8`
- Card background: `#FFFFFF`
- Border color: `#E5E7EB`

## Primary Brand Colors
- Primary Blue: `#2563EB`
- Primary Blue Dark: `#1D4ED8`
- Soft Blue Background: `#EFF6FF`

## Metric Card Icon Colors
| Card | Icon Background | Icon Color |
|---|---|---|
| Credit Given | #EFF6FF | #2563EB |
| Collected | #ECFDF5 | #16A34A |
| Collection Rate | #EFF6FF | #2563EB |
| Outstanding | #FFF7ED | #EA580C |

## Comparison Badges
### Positive
- Background: `#ECFDF5`
- Text: `#16A34A`
- Icon: Up arrow

### Negative
- Background: `#FEF2F2`
- Text: `#DC2626`
- Icon: Down arrow

### Neutral
- Background: `#F3F4F6`
- Text: `#6B7280`
- Icon: Minus

## Typography
- Main Amount: `32px / 800`
- Card Title: `16px / 600`
- Subtitle: `14px / 500`
- Comparison Badge: `13px / 700`

## Card Styling
- Border Radius: `24px`
- Padding: `20px`
- Shadow: `0 8px 24px rgba(15, 23, 42, 0.06)`
- Gap Between Cards: `16px`

## Segmented Control (Today / Week / Month)
### Active
- Background: `#2563EB`
- Text: `#FFFFFF`

### Inactive
- Background: `#FFFFFF`
- Text: `#111827`
- Border: `#E5E7EB`

## Top Overdue Customers
- Card Background: `#FFFFFF`
- Header Text: `#111827`
- Empty Icon: `#9CA3AF`
- Empty Text: `#9CA3AF`
- Overdue Amount: `#EA580C`

## Currency Rule
Use the `₹` symbol everywhere.

### Examples
- `₹4,250`
- `₹1,900`
- `₹0`

Do not use `Rs.` or `Rs`.

## Mobile Spacing
- Screen Padding: `16px`
- Card Grid Gap: `16px`
- Section Spacing: `24px`
- Bottom Navigation Safe Area: `16px`

## Approved Layout
1. Today / Week / Month segmented control
2. Four metric cards
3. Comparison percentages
4. Top Overdue Customers section
5. Empty state when no overdue customers exist

## Charts
Charts are intentionally deferred for V1.

## Resulting Style
- Consistent with the new dashboard redesign
- Modern fintech appearance
- Clear positive/negative indicators
- Better readability on mobile
- Professional card-based reporting UI
